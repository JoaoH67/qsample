GHZ repeat until success protocol

  • Repeat GHZ preparation circuit for a maximum of 3 times.
  • Only repeat if a measurement yields 1, i.e. the flag has been triggered.
  • If we measure in all 3 runs three times 1, count a logical failure.
  • In all other cases the protocol completed without logical failure.
ghz3.draw(figsize=(6,6))

tele.draw()

GHZ stabilizer measurement

  • Prepare GHZ state with flag measurement 0. (If 1 is measured repeat preparation as often as necessary)
  • Reset ancilla and measure the stabilizers of the GHZ state (without noise). The stabilizers are:
    • One XXXX stab
    • and three ZZ stabs
  • Use syndrom from stabilizer measurements to lookup and perform the correction.
  • Measure stabilizers again.
  • Now we have a logical failure if this measurement gives us any syndrome, i.e. if the measurement is anything other than four 0s.

Samplings

from qsample.sampler.direct import DirectSampler
from qsample.sampler.subset import SubsetSampler
from qsample.noise import E1
from qsample.callbacks import *

from qsample.sim.stabilizer import StabilizerSimulator as CHP
import numpy as np

import matplotlib.pyplot as plt

Use same error parameters for all examplatory samplings.

sample_range = np.logspace(-3,0,5)
err_params = {'q': sample_range}
p_max = {'q': 0.1}

GHZ repeat until success

mc_sam = DirectSampler(ghz3, CHP, err_params=err_params, err_model=E1)
mc_sam.run(20000, callbacks=[])
ss_sam = SubsetSampler(protocol=ghz3, simulator=CHP,  p_max=p_max, err_model=E1, err_params=err_params)
ss_sam.run(1000, callbacks=[])
ss_sam.tree.draw()

p_L, std = mc_sam.stats()
p_L_low, ss_std_low, p_L_up, ss_std_up = ss_sam.stats()

plt.errorbar(sample_range, p_L, fmt='--', c="black", yerr=std, label="Direct MC")
plt.loglog(sample_range, p_L_low, label='SS low')
plt.fill_between(sample_range, p_L_low-ss_std_low, p_L_low+ss_std_low, alpha=0.2)
plt.loglog(sample_range, p_L_up, label='SS low')
plt.fill_between(sample_range, p_L_up-ss_std_up, p_L_up+ss_std_up, alpha=0.2)
plt.ylabel('$p_L$')
plt.xlabel('$p$')

    'q1': sample_range,
    'q2': 0.1 * sample_range,
}
mc_sam = DirectSampler(ghz3, CHP, err_params=err_params, err_model=E2)
mc_sam.run(10000, callbacks=[PlotStats()])

ss_sam = SubsetSampler(ghz3, CHP, p_max=p_max, err_params=err_params, err_model=E2)
ss_sam.run(5000, callbacks=[PlotStats()])

p_L, std = mc_sam.stats()
p_L_low, ss_std_low, p_L_up, ss_std_up = ss_sam.stats()

plt.errorbar(sample_range, p_L, fmt='--', c="black", yerr=std, label="Direct MC")

plt.loglog(sample_range, p_L_low, label='SS low')

plt.yscale('log')
